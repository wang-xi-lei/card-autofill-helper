## 📖 使用说明

### 生成单张卡号

1. **选择BIN**
   - 从下拉菜单选择预设BIN
   - 或在"自定义BIN"输入框输入自己的BIN（优先级更高）

2. **点击"生成单张卡号"**
   - 卡片信息会显示在下方的卡片预览区域

3. **复制卡号**
   - 点击"复制卡号"按钮
   - 格式：`卡号|有效期|CVV`

### 批量生成卡号

1. **配置生成参数**
   - 选择BIN卡头
   - 选择生成数量（5/10/20/50/100）

2. **点击"批量生成"**
   - 结果显示在文本框中
   - 包含生成统计信息

3. **复制或清空**
   - "复制全部"：复制所有生成的卡号
   - "清空"：清空文本框内容

## 🛠️ 技术实现

### 核心算法
- **Luhn算法**：用于生成和验证卡号校验位
- **卡品牌识别**：根据BIN前缀自动识别卡品牌
- **随机生成**：使用安全的随机数生成账户号段

### 技术栈
- 纯HTML + CSS + JavaScript
- 无外部依赖
- 响应式设计
- 单文件应用（约60KB）

## 💡 使用技巧

### 自定义BIN优先级
自定义BIN输入框的优先级**高于**下拉选择：
```
✅ 如果填写了自定义BIN → 使用自定义BIN
❌ 如果未填写自定义BIN → 使用下拉选择的BIN
```
